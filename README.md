# SNH48 Live Enhancer

> 优化 SNH48 GROUP 公演直播网站（live.48.cn）体验的 Chrome 浏览器扩展。

## 版本

- **当前版本**：2.1.0
- **Manifest 版本**：MV3
- **适用站点**：<https://live.48.cn/*>

## 主要功能

| 功能 | 说明 |
|------|------|
| 暗黑模式 | 全站元素暗色化；公演贡献榜、成员卡片、列表页、播放器、输入框等均适配 |
| 视频增强 | 视频播放器快捷键（全屏 / 画中画 / 截图等），播放体验优化 |
| 成员搜索 | 搜索成员名即可反向检索其参与的公演；访问公演页时自动建立本地索引 |
| 嵌入搜索入口 | 页面顶部直接输入关键词，跨成员 / 公演检索 |
| 公演提醒 | 即将开始的公演桌面通知，避免错过直播 |
| 快捷导航 | 页面底部常驻导航栏，快速切换常用功能页 |
| 头部隐藏 | 观看页面可一键折叠顶部导航，腾出更多画面 |

## 安装（开发者模式）

1. 打开 Chrome 扩展页面：`chrome://extensions/`
2. 右上角开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本目录（`release-v2.1.0/`）
5. 访问 <https://live.48.cn/> 开始使用

## 安装（发布包）

已打包的发行包位于项目根目录：`SNH48-Live-Enhancer-v2.1.0.zip`。
可直接拖入 `chrome://extensions/`，或上传至 Chrome 网上应用店。

## 权限说明

| 权限 | 用途 |
|------|------|
| `storage` / `unlimitedStorage` | 保存用户配置与成员-公演索引 |
| `notifications` | 公演开始前推送桌面通知 |
| `alarms` | 精确时间触发提醒 |
| `activeTab` | 注入快捷操作所需当前标签页上下文 |
| `tabs` | 配置变更后同步到已打开的 live.48.cn 标签 |
| `contextMenus` | 右键菜单索引功能 |
| `https://live.48.cn/*` | 插件唯一作用域；不会访问其他站点 |

本扩展仅在 `live.48.cn` 生效，不收集、不上传任何用户数据，索引存储于本地。

## 常见问题

**Q：为什么打开 live.48.cn 后没有出现插件 UI？**
A：首次加载需要等待 `document_idle` 就绪。请刷新一次页面，或检查扩展页上本扩展是否启用。

**Q：成员搜索找不到某场公演？**
A：索引是在你访问公演页时建立的。点击扩展弹窗中的「清空 / 重建索引」可以手动触发一次扫描，也可通过指定 `id` 范围批量建立索引。

**Q：暗黑模式下某元素仍为亮色？**
A：如遇到样式漏覆盖，欢迎在仓库提 Issue 附该元素的截图与 DOM 类名。
